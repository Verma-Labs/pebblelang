# PebbleLang

A simple, human-readable programming language (...baby steps).

## Example Syntax

set x to 5
increase x by 2
decrease x by 1
show x

## How to Run

python pebble.py example.peb

## Supported Commands

- `set x to 5` → create a variable
- `increase x by 2` → add to it
- `decrease x by 1` → subtract
- `show x` → print it
- `comment: ...` → ignored

Suggestions welcome.
